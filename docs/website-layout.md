# Описание макета веб-сайта


## Общая структура

Веб-сайт имеет минималистичный макет:
- **Фиксированная верхняя навигационная панель (шапка)** по всей ширине, но элементы навигации не растянуты, а имеют отступ от края. Первый элемент навигации оставлен для логотипа и подписи сайта, дальше идут элементы навигации:
  - Книги 
  - Авторы 
  - Шкаф
- **Основной контент** под шапкой для станицы книг, разделён на две колонки: левая боковая панель (поисковая строка, список основных тэгов, возможность открыть еще тэги) и широкая правая колонка с контентом (книги).

---

## Основной контент страницы "книги"

### Верхняя секция (заголовок)
- **Заголовок:** Библиотека.
  - Крупный, жирный шрифт.
- **Выравнивание:** логотип и текст по левому краю контентной области.

### Две колонки под заголовком

#### 1. Левая боковая панель (Sidebar) — «КОЛЛЕКЦИИ»

- **Поисковая строка**. Значок лупы с предложением ввести текст, после ввода справа появляется кнопка отправление запроса.
- **Заголовки:** "Жанры", "Тэги" со стрелкой, открывающей выпадающие списки
- **Список категорий:** вертикальный список появляется при нажатии на заголовки или стрелки рядом с ними. делается соответствующий запрос на сервер для получения списка "основных тэгов", "основных жанров"
- **Ширина:** примерно 1/4 или 1/5 ширины основного контента. Выравнивание по левому краю.

#### 2. Правая колонка контента

- **Фон:** такой же, как у верхней секции.
- **Список элементов:** вертикальный список (статьи/книги), каждый на всю ширину колонки. Между элементами — тонкая горизонтальная линия.
- **Структура одного элемента:**
  - **Слева:** изображение (обложка книги) — фиксированные ширина и высота.
  - **Справа от изображения:** текстовый блок на оставшуюся ширину.
    - **Заголовок (название):** крупный шрифт.
    - **Автор:** меньше по размеру, заглавные буквы.
    - **Описание/аннотация:**
    - **Тэги** тэги выделяются в рамки, занимают максимум 1 строку, все тэги книги, которые не помещаются можно нажатием на крайний поместившийся тэг раздвинуть и тогда будут отрисованы все тэги в несколько строк при необходимости
    - **Жанры** аналогично тэгам

---

## Общие принципы

- **Простота:** минималистичный дизайн, много «воздуха».
- **Иерархия:** размер, начертание и цвет выделяют важные элементы.
- **Колонки:** боковая панель + основная контентная область (типично для блога/каталога).
- **Шапка:** предполагается фиксированной при прокрутке.
